Integração com Slack

Converse com seus usuários diretamente do Slack.

Adicione o Chatz BOT no seu Workspace

Para que seja possível interagir com seus usuários diretamente do Slack, é necessário integrar seu Workspace com o Chatz BOT.

Conecte com sua conta Slack nos dar permissão de interagir com seu Workspace.

Em seguida vc pode escolher o canal que será utilizado para que receba as primeiras mensagens dos usuário.

Você pode utilizar normalmente as funções do Slack para dividir o atendimento com as pessoas do seu time.

Converse com seus usuários

Quando um usuário inicia uma conversa, é criado um canal do Slack exclusivo para atendê-lo.

O nome do canal tem a inicial "chz", como por exemplo, "chz-sandro-simas".

Para enviar uma mensagem ao usuário, basta utilizar o comando "/chz <mensagem>".

Perfil do usuário

As informações do perfil do usuário são enviadad toda vez que o usuário inicia uma conversa.

Você também pode consultar o perfil através do comando "/profile".
