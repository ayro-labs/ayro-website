Integração com Websites

Integre o Chatz no seu Website

Usando tags script

1º Passo - Inclua a biblioteca

Inclua a biblioteca inserindo o código a seguir no final da tag body do HTML da página desejada.

<script type="text/javascript" src="https://cdn.chatz.io/js/chatz-1.0.0.js"></script>

Você também pode optar por baixar a biblioteca (aqui) e modificar o código acima para referenciar a biblioteca baixada.

2º Passo - Inicialize a biblioteca

Inicialize a biblioteca inserindo o código a seguir no final da tag body do HTML da página desejada, após o código inserido no passo anterior.

<script>
  Chatz.init({{ '{' }}app_token: 'token'{{ '}' }});
</script>

O token pode ser obtido acessando a página inicial do seu App.

Customizações

Você pode customizar o Chatz para que ele fique com a aparência do seu Website.

Aparência do Widget

Ao acessar a tela de configurações da integração com Websites, você consegue customizar as cores de fundo do botão flutuante, cabeçalho da caixa de chat e mensagens dos usuários.

Textos

Português do Brasil é a língua padrão utilizada nos textos que aparecem na caixa de chat. Você pode alterar os textos padrões definindo propriedades na inicialização do Chatz.

Chatz.init({{ '{' }}
  app_token: 'token',
  chatbox: {{ '{' }}
    // Título da caixa de chat
    title: 'Como podemos ajudá-lo?',
    // Placeholder do campo de texto da mensagem
    input_placeholder: 'Digite uma mensagem...'
  {{ '}' }}
{{ '}' }});
