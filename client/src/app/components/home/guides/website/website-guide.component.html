Integração com Websites

Deixe seus usuários conversarem com sua equipe diretamente do seu Website.

Instalação

Usando tag script

1º Passo - Inclua a biblioteca

Inclua a biblioteca inserindo o código a seguir no final da tag body do HTML da página desejada.

<script type="text/javascript" src="{{ sdkUrl }}"></script>

Você também pode optar por baixar a biblioteca (aqui) e modificar o código acima para referenciar a biblioteca baixada.

2º Passo - Inicialize a biblioteca

Inicialize a biblioteca inserindo o código a seguir no final da tag body do HTML da página desejada, após o código inserido no passo anterior.

<script>
  Ayro.init({{ '{' }}app_token: '<app_token>'{{ '}' }});
</script>

O token pode ser obtido acessando a página inicial do seu App.

Usando npm e browserify/webpack

1º Passo - Instale a biblioteca
npm install ayro@{{ sdkVersion }}
2º Passo - Importe a biblioteca
// Usando require
const Ayro = require('ayro');
// Usando import
import * as Ayro from 'ayro';
3º Passo - Inicialize a biblioteca
Ayro.init({{ '{' }}app_token: '<app_token>'{{ '}' }});

O token pode ser obtido acessando a página inicial do seu App.

Customizações

Você pode customizar o Chat para que ele fique com a aparência do seu Website.

Aparência do Widget

Ao acessar a tela de configurações da integração com WordPress, você consegue customizar as cores de fundo do botão flutuante, cabeçalho da caixa de chat e balão da mensagem.

Textos

Português do Brasil é a língua padrão utilizada nos textos que aparecem na caixa de chat. Você pode alterar os textos padrões definindo propriedades na inicialização do Ayro.

Ayro.init({{ '{' }}
  app_token: '<app_token>',
  chatbox: {{ '{' }}
    // Título da caixa de chat
    title: 'Como podemos ajudá-lo?',
    // Placeholder do campo de texto da mensagem
    input_placeholder: 'Digite uma mensagem...'
  {{ '}' }}
{{ '}' }});
