Informações do usuário

Conhecer o usuário te ajuda a oferecer um suporte mais eficiente e direcionado.

A biblioteca do Ayro para Websites e aplicativos te permite registrar informações do usuário. Essas informações serão repassadas no canal da empresa, quando o usuário iniciar uma conversa.

Informações genéricas

Biblioteca Javascript

1º Passo - Definição do usuário

Crie um objeto javascript e defina os atributos do usuário.

var user = {{ '{' }}
  uid: 'identificador único',
  first_name: 'primeiro nome',
  last_name: 'último nome',
  email: 'email',
  picture_url: 'url da imagem do usuário',
  signup_date: new Date()
{{ '}' }};
2º Passo - Envio de informações

Envie as informações do usuário no momento do login.

Ayro.login(user);

Você também pode atualizar as informações em qualquer outro momento, após o login do usuário.

Ayro.updateUser(user);

Biblioteca Android

1º Passo - Definição do usuário

Crie uma instância da classe User e utilize os métodos setters para atribuir as informações do usuário.

User user = new User();
user.setUid('identificador único');
user.setFirstName('primeiro nome');
user.setLastName('último nome');
user.setEmail('email');
user.setPictureUrl('url da imagem do usuário');
user.setSignUpDate(new Date());
2º Passo - Envio de informações

Envie as informações do usuário no momento do login.

Ayro.login(user);

Você também pode atualizar as informações em qualquer outro momento, após o login do usuário.

Ayro.updateUser(user);

Informações adicionais

Além das informações genéricas, você pode associar qualquer informação adicional ao usuário. Para isso, basta utilizar a propriedade user.properties.

Biblioteca Javascript

var user = {{ '{' }}
  properties: {{ '{' }}
    age: '18',
    gender: 'male'
  {{ '}' }}
{{ '}' }};
Ayro.updateUser(user);

Biblioteca Android

Map<String, String> properties = new HasMap<>();
properties.put('age', '18');
properties.put('gender', 'male');
User user = new User();
user.setProperties(properties);
Ayro.updateUser(user);
