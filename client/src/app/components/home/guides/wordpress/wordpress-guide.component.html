Integração com WordPress

Deixe seus usuários conversarem com sua equipe diretamente do seu website WordPress.

Instale o plugin

Faça o download do plugin (aqui) e extraia o conteúdo do arquivo no diretório de plugins do WordPress em wp-content/plugins.

Você também instalar o plugin através do painél de administração do WordPress, na tela Plugins > Adicionar Novo, buscando por Ayro.

Em seguida basta acessar a tela Plugins > Plugins Instalados e ativar o plugin Ayro.

Configure o plugin

Ao ativar o plugin uma nova tela de configuração aparecerá em Configurações > Ayro.

Acessando esta tela, é possível configurar o plugin Ayro.

Insira o token, que pode ser obtido acessando a página inicial do seu App, no campo App Token e, opcionalmente, altere a tradução dos textos exibidos no Widget do Chat.

Customizações

Você pode customizar o Chat para que ele fique com a aparência do seu website Wordpress.

Aparência do Widget

Ao acessar a tela de configurações da integração com WordPress, você consegue customizar as cores de fundo do botão flutuante, cabeçalho da caixa de chat e balão da mensagem.

Textos

Português do Brasil é a língua padrão utilizada nos textos que aparecem na caixa de chat. Você pode alterar os textos padrões na tela de configuração do plugin em Configurações > Ayro.
