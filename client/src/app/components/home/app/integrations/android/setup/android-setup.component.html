1º Passo - Instale a biblioteca

Adicione a biblioteca Android do Chatz como dependência do seu projeto

Usando Gradle:

compile 'io.chatz:chatz:1.0.0'

Usando Maven:

<dependency>
  <groupId>io.chatz</groupId>
  <artifactId>chatz</artifactId>
  <version>1.0.0</version>
</dependency>
2º Passo - Inicialize a biblioteca

Inicialize a biblioteca incluindo o código a seguir no método onCreate da sua classe que estende Application.

ChatzIO.init(this, "{{ app.token }}")
