1º Passo - Baixe a biblioteca

Baixe a biblioteca Javascript do Chatz para Websites aqui.

2º Passo - Importe a biblioteca

Importe a biblioteca incluindo o código a seguir no final da tag body do HTML da página desejada.

<script type="text/javascript" src="/caminho/para/chatz.js"></script>
3º Passo - Inicialize a biblioteca

Inicialize a biblioteca incluindo o código a seguir no final da tag body do HTML da página desejada, após o código inserido no passo anterior.

<script>
    ChatzIO.init({{ '{' }}
      app_token: '{{ app.token }}'
    {{ '}' }});
  </script>
4º Passo - Teste a integração

Rode o seu Website depois de ter completado todos os passos anteriores e nos deixe saber se sua integração está funcionando corretamente clicando no botão abaixo.
